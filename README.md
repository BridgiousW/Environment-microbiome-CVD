# Link between Helminths, Environment, and the gut microbiome, and how these all relate with CVD

The role of microbiome and metabolome in modulating the effect of helminths and  environment on Cardiovascular disease risk

The scripts and files in the `root` directory are for reproducing the analyses and figures in the pre-print https://www.researchsquare.com/article/rs-7108139/v1
